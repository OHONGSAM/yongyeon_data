## 0. 약어 용어집

본 문서에 등장하는 주요 대문자 약어의 전체 명칭과 의미입니다.

| 약어 | 전체 명칭 | 설명 |
|------|-----------|------|
| **RCS** | Remote Control Substation (원격제어 소국) | 정수장 현장 각 계측 지점의 센서 데이터를 수집·전송하는 원격 제어 서브스테이션. `RCS_6`, `RCS_3` 등의 번호는 해당 소국의 식별 번호. SCADA 시스템의 하위 구성 요소. |
| **FCC** | Flow Control Computer (유량 제어 컴퓨터) | 유량 계측값을 기반으로 약품 주입량을 비례 제어하는 현장 제어 장치. `RCS_3.FCC_19`는 RCS_3 소국 산하 19번 FCC 장치를 의미함. |
| **AI** | Analog Input (아날로그 입력) | 센서로부터 연속적인 아날로그 신호를 읽어 들이는 계측 채널. 컬럼명의 `.AI.`는 아날로그 센서 실시간 계측값임을 나타냄. |
| **PPM** | Parts Per Million | 농도 단위. 100만분의 1을 의미하며 수처리에서는 mg/L와 실질적으로 동일하게 사용됨. |
| **PH** | pH (수소이온농도지수) | 수용액의 산성·알칼리성 정도 (0~14). 정수 기준 5.8~8.5 관리. |

> **컬럼명 읽는 법 예시:** `RCS_6.AI.침전지1_잔류염소`
> → **RCS_6** (6번 원격제어 소국) **.AI.** (아날로그 입력 채널) **침전지1_잔류염소** (측정 항목)
>
> **또 다른 예:** `RCS_3.FCC_19.AI.통합여과수_잔류염소`
> → **RCS_3** (3번 소국) **FCC_19** (19번 유량제어 컴퓨터) **.AI.** (아날로그 입력) **통합여과수_잔류염소** (측정 항목)
