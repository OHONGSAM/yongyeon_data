## 0. 약어 용어집

본 문서에 등장하는 주요 대문자 약어의 전체 명칭과 의미입니다.

| 약어 | 전체 명칭 | 설명 |
|------|-----------|------|
| **PAC** | Poly Aluminum Chloride (폴리염화알루미늄) | 정수 처리에서 사용하는 무기 고분자 응집제. 원수의 탁도 유발 미립자를 응집·침전시키는 데 사용됨. `PAC.AI.*` 컬럼은 PAC 주입 시스템의 계측값. |
| **PACS** | PAC System 2 (2계열 PAC 시스템) | 덕남 정수장 PAC 주입 2계열 시스템을 구분하기 위한 명칭. `PACS2`는 2호 계열을 의미. |
| **RCS** | Remote Control Substation (원격제어 소국) | 정수장 현장 각 계측 지점의 센서 데이터를 수집·전송하는 원격 제어 서브스테이션. `RCS_6`, `RCS_1` 등의 번호는 해당 소국의 식별 번호. |
| **AI** | Analog Input (아날로그 입력) | 센서로부터 연속적인 아날로그 신호를 읽어 들이는 계측 채널. 컬럼명의 `.AI.`는 아날로그 센서 실시간 계측값임을 나타냄. |
| **SV** | Set Value (설정값) | 제어 시스템에서 운전자가 지정한 목표값(설정치). 실제 계측값(PV, Present Value)과 대비됨. |
| **FT** | Flow Transmitter (유량 전송기) | 유량을 측정하여 신호로 전송하는 계기. `FT101`은 착수정 유입유량계를 의미. |
| **LT** | Level Transmitter (수위 전송기) | 수위를 측정하여 신호로 전송하는 계기. `LT301`, `LT302`는 각 수위계의 태그 번호. |
| **TB** | Turbidity (탁도) | 물의 탁함 정도. 컬럼명 `착수정_TB`에서 TB는 Turbidity의 약자. 단위 NTU. |
| **AL** | Aluminum (알루미늄 농도) | 응집제 투입 후 수중 잔류 알루미늄 농도 계측값. |
| **PH** | pH (수소이온농도지수) | 수용액의 산성·알칼리성 정도 (0~14). 정수 기준 5.8~8.5 관리. |

> **컬럼명 읽는 법 예시:** `RCS_6.AI.착수정_TB`
> → **RCS_6** (6번 원격제어 소국) **.AI.** (아날로그 입력 채널) **착수정_TB** (착수정 탁도)
